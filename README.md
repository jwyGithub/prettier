### install

#### with pnpm

```sh
pnpm add @jiangweiye/prettier-config -D
```

#### with yarn

```sh
yarn add @jiangweiye/prettier-config -D
```

#### with npm

```sh
npm install @jiangweiye/prettier-config -D
```
